# Задание 3

## Задача 1

### Текст задачи

Вывести пути ко всем текстовым файлам (\*.txt) в указанном каталоге и

его подкаталогах



### Алгоритм решения

1\. Получение пути к каталогу

Сначала проверяются аргументы командной строки. Если путь передан и существует – он берётся.



Если нет (аргумента нет или путь неверен), включается интерактивный режим: программа запрашивает ввод, удаляет возможные внешние кавычки и проверяет, существует ли такая директория. При ошибке запрос повторяется.



2\. Обход каталогов

Функция findTxtFiles рекурсивно обходит все папки, начиная с заданной.



В каждой папке с помощью Directory.GetFiles собираются все файлы с расширением .txt и добавляются в последовательность.



Затем для каждой подпапки (через Directory.GetDirectories) рекурсивно вызывается та же функция, и результаты объединяются (yield!).



3\. Вывод результатов

Полученная последовательность путей преобразуется в список (чтобы можно было проверить пустоту).



Если список пуст – выводится сообщение «Файлы .txt не найдены».



Иначе каждый путь выводится на отдельной строке.
